Witaj, nazywam się Jarosław Terejko i jestem początkującym Junior Front-end Developerem

Swoje pierwsze "Hello World" popełniłem w połowie 2017 roku i od tamtego czasu systematycznie, małymi krokami, w domowym zaciszu zacząłem zgłębiać tajniki tworzenia front-endu.

Warstwę wizualną aplikacji koduję zgodnie z najnowszych standardami HTML5 oraz CSS3. Znajomość podstaw JavaScript i jQuery pozwala mi dodawać inteaktywność do moich projektów, co zwiększa aktrykcyjność tworzenego UI oraz UX.

Znam i praktycznie wykorzystuję Bootstrap, najpopularniejszy framework CSS. Nie jest mi również obca koncepcja kontroli wersji, którą w praktyce stosuję przy pomocy Git. Automatyzację powtarzalnych zadań powierzam task runnerowi Grunt.
